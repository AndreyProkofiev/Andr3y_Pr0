## curriculum vitae
На текущем месте работы

**Разработка AI-сервисов (CV, NLP, LLM).**

- Реализовал сервис автоматизации технической поддержки, сокративший среднее время реакции на заявки с **15 дней до 1,2 дня**. Система автоматически определяет по тексту заявки, в какую очередь и команду направить тикет для решения проблемы.
- В составе кросс-функциональной команды разработал Telegram-бота с ИИ, который:
    - консультирует пользователей по техническим вопросам;
    - выполняет удалённые действия (перезагрузка компьютера, сброс пароля и др.).
- Разработал рекомендательную систему для корпоративного портала.
- Создал сервис автоматического обновления базы знаний чат-ботов.
- Имею опыт построения решений на основе:
    - **RAG**, **LLM**, **VLM**, **OCR**;
    - классического машинного обучения (бустинг-алгоритмы, логистическая регрессия и др.);
    - компьютерного зрения (например, фильтр Лапласа);
    - NLP (TF-IDF, векторизация текстов и др.)
