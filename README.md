Реализовал сервис автоматизации технической поддержки, сокративший среднее время реакции на заявки с  5 дней до 1,2 дня. 

Система автоматически определяет по тексту заявки, в какую очередь и команду направить тикет для решения проблемы.

Использовал комбинацию классических NLP-признаков (TF-IDF) и логистическая регрессии, плюс KNN на эмбеддингах модели multilingual-e5-base.
Продуктивизировал в виде FastAPI-сервиса с логированием и мониторингом.

В составе кросс-функциональной команды разработал Telegram-бота с ИИ, который:
    - консультирует пользователей по техническим вопросам (моя часть разработка RAG сервиса с API);
    - выполняет удалённые действия (перезагрузка компьютера, сброс пароля и др.)( моя часть разработка классификатора интента пользователя)

Разработал рекомендательную систему для корпоративного портала.
Сервис оффлайн рекомендаций, продуктивизирован в виде Airflow DAG.

Создал сервис автоматического обновления базы знаний чат-ботов.
Airflow DAG, запускается ежедневно:
- проверяет наличие изменений в Confluence.
- при обнаружении изменений, векторизует текст статей.
- добавляет в векторную базу RAG бота.
- формирует отчет о проделанной работе.
